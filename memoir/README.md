# Memoir

Theme for [Hexo].

## Install

Execute the following command and modify `theme` in `_config.yml` to `memoir`.

```
git clone git://github.com/yhben/hexo-theme-memoir.git themes/memoir
```

## Update

Execute the following command to update Light.

```
cd themes/memoir
git pull
```
